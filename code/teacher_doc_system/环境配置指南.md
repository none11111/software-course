~ 教师文档管理系统 - 环境配置指南

~~ 🚀 快速配置步骤

~~~ 第一步：环境准备

1. 安装Python 3.8+
   - 下载地址：https://www.python.org/downloads/
   - 安装时勾选"Add Python to PATH"

2. 安装MySQL 5.7+
   - 下载地址：https://dev.mysql.com/downloads/mysql/
   - 记住设置的root密码

3. 安装Git（可选）
   - 下载地址：https://git-scm.com/downloads

~~~ 第二步：获取项目代码

方式一：直接下载
- 将整个项目文件夹复制到新电脑

方式二：Git克隆
```bash
git clone <项目地址>
cd teacher_doc_system
```

~~~ 第三步：安装依赖

1. 打开命令行
   - Windows：按 `Win + R`，输入 `cmd`
   - 进入项目目录：`cd 项目路径`

2. 安装Python依赖
```bash
pip install -r requirements.txt
```

~~~ 第四步：配置数据库

1. 创建数据库
   - 打开MySQL命令行或图形界面工具
   - 执行以下命令：
```sql
CREATE DATABASE teacher_doc_system CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

2. 配置环境变量
   - 复制 `env.example` 文件并重命名为 `.env`
   - 编辑 `.env` 文件，修改数据库配置：
```env
~ 数据库配置
DB_NAME=teacher_doc_system
DB_USER=root
DB_PASSWORD=你的MySQL密码
DB_HOST=localhost
DB_PORT=3306

~ Django配置
SECRET_KEY=your-secret-key-here
DEBUG=True
ALLOWED_HOSTS=localhost,127.0.0.1,你的IP地址
```

~~~ 第五步：初始化系统

1. 初始化数据库
```bash
python setup_database.py
```

2. 启动系统
```bash
~ Windows用户
双击运行 start_lan.bat

~ 或者命令行启动
python start.py --lan
```

~~~ 第六步：访问系统

1. 打开浏览器
   - 访问：http://localhost:8000
   - 或局域网访问：http://你的IP:8000

2. 登录系统
   - 管理员账号：admin
   - 管理员密码：Admin123456

~~ 🔧 常见问题解决

~~~ 问题1：pip安装失败
```bash
~ 升级pip
python -m pip install --upgrade pip

~ 使用国内镜像
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple/
```

~~~ 问题2：MySQL连接失败
- 检查MySQL服务是否启动
- 确认数据库用户名和密码正确
- 检查防火墙设置

~~~ 问题3：端口被占用
```bash
~ 查看端口占用
netstat -ano | findstr :8000

~ 停止占用进程
python stop.py
```

~~~ 问题4：无法局域网访问
- 检查防火墙设置
- 确认IP地址正确
- 使用 `python start.py --lan` 启动
