在同一局域网的其他设备上打开浏览器，访问：
```
http://[您的IP地址]:8000
```
例如：`http://192.168.1.100:8000`

-- 🛠️ 故障排除

--- 无法访问？
1. 检查防火墙
   - Windows: 允许Python通过防火墙
   - 确保端口8000未被阻止

2. 检查网络
   ```bash
   - 测试网络连接
   python test_network.py
   ```

3. 检查端口占用
   ```
   - Windows
   netstat -ano | findstr :8000
   

--- 常见问题
- 端口被占用: 关闭其他使用8000端口的程序
- 防火墙阻止: 添加Python到防火墙例外
- IP地址错误: 使用 `ipconfig` (Windows)查看正确IP
